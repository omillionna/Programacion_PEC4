# Programacion_PEC4
PEC4 – Programación para la Ciencia de Datos (UOC). Proyecto en Python para el análisis del rendimiento y abandono universitario en Cataluña, con estructura modular, visualización, testing y documentación.

## Estructura del proyecto
- main.py: punto de entrada del proyecto
- src/: código fuente
- tests/: tests unitarios
- doc/: documentación generada
- screenshots/: capturas requeridas en la PEC
